# A Learning-Based Closed-Loop Fluid Flow Regulation

This repository contains code and resources related to the research project:

**A Learning-Based Closed-Loop Fluid Flow Regulation**

## Overview

The objective of this work is to develop and validate a learning-based control framework for the closed-loop fluid flow regulation. The repository includes control algorithms, simulation models, experimental data, and documentation required to reproduce the reported results.
